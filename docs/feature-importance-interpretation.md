# Feature Importance Interpretation

이 문서는 Colab 실행 결과 `data/colab/feature_importance.csv`의 상위 feature를 해석합니다.

## Summary

모델이 가장 중요하게 본 것은 크게 세 가지입니다.

1. 직전/현재 시간대의 이동 흐름
2. 시간대 자체
3. 행정동의 구조적 특성

즉, 모델은 “지금 사람이 어디로 얼마나 이동하고 있는가”, “현재가 몇 시인가”, “그 동이 어떤 공간적 성격을 갖는가”를 보고 1시간 뒤 이동 수요 proxy를 예측합니다.

## What This Means

상위 feature가 대중교통 OD와 시간대에 집중되어 있다는 것은 자연스럽습니다.

현재 target 자체가 `1시간 뒤 인구 1,000명당 유입 이동 수요 proxy`이기 때문에, 현재의 유입/유출 흐름과 시간대가 가장 강하게 작동합니다.

POI/건물 feature는 순간 변동을 직접 만드는 feature라기보다, 동별 기본 성격을 구분하는 역할을 합니다. 예를 들어 호텔 수, 건물 높이, 동 면적은 “이 동이 업무/상업/방문 수요가 강한 곳인지”를 설명합니다.

## Presentation Wording

발표에서는 이렇게 말하면 안전합니다.

> Feature importance를 확인해보면, 모델은 현재 시간대의 대중교통 유입/유출 흐름과 시간대를 가장 중요하게 사용했습니다. 이는 강남권 이동 수요가 시간대와 현재 이동 흐름에 크게 좌우된다는 점과 일치합니다. 여기에 건물 높이, 호텔 수, 동 면적 같은 POI/도시 구조 feature가 동별 기본 수요 차이를 보완했습니다.

또는 더 짧게:

> 모델은 단순히 동 이름만 외운 것이 아니라, 현재 이동 흐름, 시간대, 도시 구조 feature를 함께 사용해 1시간 뒤 이동 수요 proxy를 예측했습니다.

## Caution

이 결과를 다음처럼 말하면 안 됩니다.

